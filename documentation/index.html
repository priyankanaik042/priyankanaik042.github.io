Marqa is Html5 + CSS3 portfolio template and works fine in all major browsers. It's powered by jQuery and you'll find nice and discreet interactivity. The code is clearly written and you will find comments for each section parts. Lets take a closer look at the structure of Html, Css, JavaScript...

Folders and Files

When you unpack downloaded archive from themeforest.net you will find it as the following: 

  1. Documentation Folder:  Documentation for the convertible site template.
  2. Marqa Folder:  Main folder of the template ( which contains folders and files of the template ).

Marqa Folder Structure

- When you open Marqa folder you will find : 

  1.  CSS folder   :  contains all CSS files used in the template.
  2.  Fonts folder :  contains all fonts used in the template.
  3.  img folder     :  contains images used in the template.
  4.  Js folder       :  contains all JavaScript files used in the template.
  5.  index.html    :  main HTML file of the template.

HTML Structure

Marqa template is well commented to easily understand,

 in index.html you will find: 

      <head> area in which you can find all the meta tags and css files links.

     <body>  area which contains the site and you will find that : 

- the site is divided into sections, every section is a div with a class of its name and well commented so you can edit any data you want easily.

- Sections are sorted as their order in the site, so you can find any section you want easily.

- Most of the sections has a container div ( Bootstrap Class ) to wrap the content and make the site looks better, and every section is divided into smaller parts

<!-- Start Header Section -->
<div class="header" id="header">

    <div id="large-header" class="large-header">

        <canvas id="demo-canvas"></canvas>

        <div class="content center">

            <div class="container">

                <p class="wow fadeInDown" data-wow-duration="0.5s">Hello, It's me</p>

                <h1 class="wow fadeInDown" data-wow-duration="0.5s" data-wow-delay=".5s">John Doe</h1>

                <p class="wow fadeInDown" data-wow-duration="0.5s" data-wow-delay="1s">ui/ux designer - developer</p>

                <a href="#about-me" class="wow fadeIn" data-wow-duration="0.5s" data-wow-delay="1.5s">Read more</a>

                <a href="#portfolio" class="wow fadeIn" data-wow-duration="0.5s" data-wow-delay="1.5s">my works</a>

            </div> <!-- End container -->

        </div>

        <div class="mouse" id="mouse"></div> <!-- Mouse Icon -->

    </div>

</div>

<!-- End Header Section -->

    $('#Container').mixItUp();



    // magnific popup in portfolio section //

    $('.open-popup-link').magnificPopup({

        type:"inline",

        fixedContentPos:!1,

        removalDelay:100,

        closeBtnInside:!0,

        preloader:!1,

        mainClass:"mfp-fade"

    });
